TMAuthentication

class TMAuthentication

Perform authentication operations for Ticketmaster services, like login, logout, mfa process, obtain member details information, get a token to make authorized request to ticketmaster services.

Obtain an instance of this class using TMAuthentication.Builder.build. You can configure TMAuthentication using the available methods on the TMAuthentication.Builder.
