getToken

suspend fun getToken(source: AuthSource): String?

This method will return a previously fetched token during the login process.

Always returns a fresh token ready to use, so it means that will refresh the token automatically if the token it's expired at the moment you invoke this method.

Return

This method will return a valid Token as a string if you previously completed the login process successfully. It will return null if you didn't complete the login process successfully or if your token it's already expired and the refresh process fails.

We recommend invoke logout and the start the process login again the obtain a valid token.

Parameters

source

AuthSource that you want to get the token.
