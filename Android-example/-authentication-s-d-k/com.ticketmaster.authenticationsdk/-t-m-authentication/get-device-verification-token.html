getDeviceVerificationToken

suspend fun getDeviceVerificationToken(memberId: String): String?

Returns a Device Verification token previously obtained through the Multifactor Authentication process (MFA), see getMFAIntent. If this is null it means you need to prompt the MFA process to authenticate the device.

Return

Device Verification token if exists, null otherwise.
