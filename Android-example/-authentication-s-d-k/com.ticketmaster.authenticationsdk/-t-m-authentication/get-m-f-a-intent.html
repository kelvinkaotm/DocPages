getMFAIntent

fun getMFAIntent(activity: Activity, memberId: String, fromSmartQueue: Boolean = false, additionalProperties: Map<String, String> = emptyMap()): Intent?

Whenever you need to use Multifactor Authentication (MFA) you can call this method to receive an Intent that you can launch to handle the process.

You can listen the result of the process onActivityResult or registerForActivityResult.

Return

Intent that will launch the activity that handles the MFA process

Parameters

activity

The activity used to construct the intent.

memberId

Member id of this user, example TMMemberInfo.HostMember.memberId

fromSmartQueue

If you're using this from smart queue.

additionalProperties

A map that can be used to override specific properties on the MFA
